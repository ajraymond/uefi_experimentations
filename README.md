This repository contains some experiments with UEFI programming.

Pre-Rrequisites
---------------------------
$ apt install mtools ovmf gcc

$ cd ../
$ git clone https://git.code.sf.net/p/gnu-efi/code gnu-efi
$ cd gnu-efi
$ make
